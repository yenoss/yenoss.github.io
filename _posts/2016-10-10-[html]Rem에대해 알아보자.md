##rem이란?

rem은 **root em** 을 의미한다. 즉 부모요소의 상대적 크기를 이야기한다.

간단히 코드로 이해해보자.

~~~
	html { font-size: 100%; } 
	p{
		font-size:2rem;
	}
	<p>hello</p>
~~~

html 에서 font-size:100% 는 기본 h4 사이즈를 갖는다. 

2rem에 의미는 h4사이즈의 2배인 폰트 크기를 가지게된다.
